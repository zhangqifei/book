                # 开博缘由

                这是一个知识急速膨胀的时代，市面上的资料多如牛毛，可资料的质量却良莠不齐，想从浩瀚的书海里找到想要的资料，并不是一件容易的事情，本网站就是为了这个目的而建立的。

                # 网站简介

                本网站是由一个爱好读书且愿意分享的书虫组成。希望通过我们的努力，可以引导广大朋友培养读书兴趣，提高阅读技能，寻找属于自己的好书，更好地享受读书之乐。推荐书目，推动阅读，我们一直在努力。

                以下是成长过程中的一些思考，在此分享给大家，希望能让大家更加了解它。

                # 关于版权

                本网站大部分来自于网络，为表示对作者的尊重，所有资料我们都会在文章结尾标明出处，如有侵权，请联系我们进行删除。同时，我们也希望转载本网站资料朋友能标注一下转自本网站。

                关于电子书资源分享，我们既不鼓励也不禁止，但还是希望大家在力所能及的范围内支持正版。

                # 关于致歉

                由于本网站是我利用业余时间维护，大家彼此也都有自己的工作和生活，加上资金精力经验等限制，可能有很多地方做的不到位，许多朋友提出的一些建设性的意见无法采纳，在此我们诚挚地向大家道歉，希望大家能够多多理解和包涵。

                # 关于未来

                本网站是利用业余时间支撑起来的，未来会怎样还是个未知数，但我们会一直努力坚持下去。

                无论是否做大，我们都不忘初心，只要它的存在能够让一些朋友获得帮助，能够让我们成长，那么它就是有意义的。

                关于未来，我们不想太多，脚踏实地的走下去，从读书及交流中获取知识和快乐，这就是它的现在和未来。
